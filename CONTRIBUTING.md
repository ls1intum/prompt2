## ü§ù How to Contribute

### Reporting Bugs
*   Check the [Issues](https://github.com/ls1intum/prompt2/issues) to see if the bug has already been reported.
*   If not, use our [Bug Report Template](https://github.com/ls1intum/prompt2/issues/new?template=bug_report.md).

### Suggesting Enhancements
*   We love new ideas! Please use our [Feature Request Template](https://github.com/ls1intum/prompt2/issues/new?template=feature_request.md) to explain your vision.
*   For large architectural changes, it's best to discuss them on Discord first.

### Pull Requests
1.  **Branch Naming**: Use descriptive names like `feature/your-feature` or `fix/your-bug`.
2.  **Stay Focused**: Keep PRs small and focused on a single change.
3.  **Update Documentation**: If you change functionality, please update the relevant files in the `docs/` folder.
4.  **Follow Conventions**: Adhere to the naming and structural conventions defined in [AGENTS.md](./AGENTS.md).
